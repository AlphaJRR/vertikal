**As a** user  
**I want to** sign up and create a profile  
**So that** I can join the platform

**Acceptance Criteria:**
- [ ] User can enter name, email, city
- [ ] User can choose handle (unique)

- [ ] Application includes handle, category, notes

**As an** admin  
**I want to** approve creator applications  
**So that** I can assign Founding 50 badges

**Acceptance Criteria:**

**Definition of Done:**

**As a** user

- Proof screenshots + test results
